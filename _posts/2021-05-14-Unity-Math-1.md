# Unity 수학
## 좌표계
- 유니티는 왼손 좌표계 (엄지 x, 검지 y, 중지 z)
- z축이 왼손 좌표계는 멀수록 +
- 로컬 좌표계
  - 모델 내부의 상대 좌표
- 월드 좌표계
  - 씬에서 물건의 위치
- 스크린 좌표계
  - 카메라에 투영 된 2D 스크린 좌표
  - 카메라의 WorldToScreenPoint로 알 수 있음
- 극 좌표계
  - (r, theta) = (r * cos(t), r * sin(t))
  - r 은 거리이므로 = (sqrt(x^2 + y^2), atan(y, x))
  - 극 좌표계로 렌즈의 왜곡을 보정 할 수 있다.

- 구면 좌표계
  - (거리, 방위각, 양각) 로 구성

## 벡터
 - 벡터의 크기 magnitude
   - (x,y,z) = sqrt(x^2 + y^2 + z^2)
   - norm, ||v|| 으로 표기
